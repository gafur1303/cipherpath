// Complete Learning Roadmap
CyberSecurity A → Z
Follow this structured path from zero to job-ready. Each step links to free resources.
01
Foundations — Networking & Linux
TCP/IP, DNS, HTTP, subnetting, Linux CLI, bash scripting, file permissions. The non-negotiable bedrock of all security work.
THM: Pre-Security Path Professor Messer (free) NetworkChuck YouTube OverTheWire: Bandit
02
Programming & Scripting
Python for automation, Bash scripting, PowerShell basics. You don't need to be a developer but must be able to read and write scripts.
Python.org tutorials (free) THM: Python Basics Automate the Boring Stuff
03
Security Fundamentals
CIA triad, cryptography, authentication, PKI, common attack types, security policies and frameworks (NIST, ISO 27001).
THM: Jr Penetration Tester SANS Cyber Aces (free) HTB: Starting Point
04
Web Application Security
OWASP Top 10, SQL injection, XSS, CSRF, SSRF, authentication flaws, API security. Practice with Burp Suite and OWASP WebGoat.
HTB: Bug Bounty Hunter THM: OWASP Top 10 PortSwigger Web Academy PicoCTF Web Challenges
05
Network Penetration Testing
Nmap, Wireshark, Metasploit, exploitation frameworks. Active Directory attacks, SMB, FTP, SSH, RDP exploitation.
HTB: Penetration Tester THM: Attacking Active Directory TCM Security YouTube Vulnhub (free VMs)
06
CTF Competitions
Apply everything learned. Participate in beginner-friendly CTFs: PicoCTF, NahamCon CTF, CTFlearn. Crypto, pwn, web, forensics, reversing.
CTFtime.org PicoCTF (beginner) CTFlearn HackTheBox CTF
07
Specialise: Red or Blue Team
Red: Advanced exploitation, AV evasion, C2, report writing. Blue: SIEM, Splunk, log analysis, incident response, threat intelligence.
HTB: Red Team Operator THM: SOC Level 1 Splunk Free Training DFIR.training
08
Certifications & Career
CompTIA Security+, eJPT (free exam vouchers available), CEH, OSCP. Build a portfolio of HTB/THM profiles, CTF writeups, and GitHub projects.
eJPT by INE (free tier) CompTIA CertMaster (trial) HTB CPTS Cert LinkedIn + GitHub portfolio
